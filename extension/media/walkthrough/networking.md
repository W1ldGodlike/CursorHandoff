## Web access

By default the server listens on **Localhost** (`127.0.0.1`) — only this machine can open the web UI.

To use your phone:

- **LAN** — Handoff settings → **Web access** → **LAN** → set a password → **Save and restart** → `http://<your-ip>:3000`
- **Custom** — Tailscale or another IP: bind **Custom**, paste the address, save

More detail: [Reach the server from another device](command:cursorHandoff.openDoc?%22docs%2Fguide.md%23remote-access%22).

Open **CursorHandoff: Open Handoff settings** → **Web access**.
